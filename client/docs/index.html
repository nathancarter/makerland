Being a Maker

Making MakerLand what you want it to be

There are many documentation pages you can read to help you learn to be a maker. They are organized roughly in the order in which it makes sense to read them. Click one to read it, or use the navigation bar at the top of each page.

Overview

READ THIS FIRST. It is a thorough tutorial on all things related to being a MakerLand Maker. When it needs to go into detail on a specific subject, it links to one of the other documents, listed below.

Teleport Command

The Teleport command lets you memorize locations in the game, then later, immediately teleport back to a location that you have memorized. This page is the complete reference on how to use the Telport command.

Database Command

The Database command lets you add new pictures, sounds, and code to the game's library of data (called a "database"). It does not let you place these types of things in the game world; that's what the World command (below) does. This page is the complete reference on how to use the Database command.

World Command

The World command lets you take items from the game database (pictures, sounds, animations, behaviors, etc.) and place them into the game world. You can only do so in an area that you have created. This page is the complete reference on how to use the World command.

Reset Command

The Reset command lets you update all the objects near you. This is only useful if you have changed some of the code loaded into those objects, and need them to reload it. This page is the complete reference on how to use the Reset command.
